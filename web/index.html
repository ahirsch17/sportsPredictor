Pipeline

From raw feeds to accountable predictions.

Async ingestion

Multiple async workers grab NFL stats, injury updates, betting lines, and NOAA weather snapshots. Each feed is normalized, deduped, and versioned so model runs can rewind any matchup state.

  • Rate-limited requests fanned out with retry queues.
  • Schema validator enforces field parity across seasons.
  • Slack alerts trigger if any feed drifts or stalls.

Model stack

Multiple lenses, one confident probability curve.

Each ensemble member focuses on a different signal family for better calibration.

Drive momentum

Gradient Boosting (XGBoost)

Feeds on rolling drive success, pressure disruption, and personnel packages.

  • Rolling 6-week window with decay weighting.
  • Auto-calibrated with isotonic regression.
  • Outputs win probability and spread edge.
Situational football

Logistic meta-ensemble

Combines weather-adjusted EPA, rest days, travel load, and late injury overrides.

  • Cross-validated with weekly backtests.
  • Handles missing data with learned priors.
  • Surface multidimensional SHAP narratives.

Analyst sandbox

Interactive CLI layer lets analysts stress-test scenarios and export custom reports.

  • Mode toggles: conservative, neutral, upset.
  • Outputs CSV, PDF, and Markdown briefs.
  • Track override history for transparency.
